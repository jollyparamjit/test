Creating New Application Role

You can create new application roles to assign specific or set of privileges to a new user. These roles reflects in the Rolesdrop-down menu in New User dialog box.

To create a new user:

  1. Click . The following New Application Role dialog box is displayed.

  2. Specify values in each field:

    • Role Name- Unique name for the role.

    • Application Privileges- Select the application privilege fields as required. You can select multiple privilege for a single role.

      Use the following table to understand the default application privileges:

      Application Privilege Name Description
      User Management With this, the user has the rights to access User Management application and add or edit or delete any application roles.
      User Distribution With this, the user has access to User Distribution tab.
      Data Extraction With this, the user has access to Data Extraction tab.
      Trending Monitoring With this, the user has access to Behavior Trending tab.
      Comparative Breakdown With this, the user has access to Comparative Breakdown tab.
      Imports With this, the user can Import the subscribers list in the segment module.
      Exports With this, the user can Export the subscribers list in the segment module.
      Reports With this, the user has access to Reports tab.

      Note:* indicates a mandatory field.

  3. Click Create to create a new application role or Cancel to discard them.
